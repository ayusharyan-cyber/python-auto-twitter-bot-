# Python Auto Twitter Bot

## Overview
This project is a beginner-level Python application that demonstrates how to interact with the X (Twitter) API using the Tweepy library and OAuth authentication.

## Features
- Authenticate with X (Twitter) using OAuth 1.0
- Connect to Twitter API using Tweepy
- Demonstrates how tweet posting works via API
- Handles real-world API permission errors

## Tech Stack
- Python
- Tweepy
- X (Twitter) Developer API

## Important Note
Tweet posting via `update_status()` requires elevated API access.
On free/basic X developer plans, write permissions may be restricted, which can result in a 403 Forbidden error.
This project is created for learning and demonstration purposes.

## Learning Outcomes
- Using external APIs with Python
- OAuth authentication flow
- Secure handling of API keys
- Debugging real-world API errors
- Understanding API access limitations
